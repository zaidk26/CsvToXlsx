Usage:

Call the program from the CMD.

Enter file path:Desired Sheet Name and lastly the file name

e.g
================================================
1 file
CsvToXlsx.exe "C:\temp-copies\test.csv:sheet 1" "C:\temp-copies\combined.xlsx"

===============================================
multiple files
CsvToXlsx.exe "C:\temp-copies\test.csv:sheet 1"  "C:\temp-copies\Bulk Delivery Note.csv:sheet 2" "C:\temp-copies\Orders.csv:Orders"  "C:\temp-copies\combined.xlsx"
